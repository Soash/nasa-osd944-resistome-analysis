# NASA OSD944 Low-Biomass Cleanroom Resistome Profiling

This repository contains the custom bioinformatics, data-wrangling, and statistical analysis pipeline used to profile the microbiome and resistome of spacecraft assembly cleanrooms. The pipeline processes short-read Illumina metagenomic sequencing data sourced from the NASA Open Science Data Repository (OSDR).

The study maps functional gene annotations against the KEGG Antimicrobial Resistance (AMR) database and the BacMet Biocide and Metal Resistance Gene (BRG) database to identify mechanisms of microbial persistence in extreme, highly sterilized environments.

## Data Source
The primary dataset utilized in this pipeline is **OSD-944 (GeneLab ID: GLDS-775)**: *JPL Mars Exploration Program Interlab Cleanroom Metagenomics*. 
* **Link:** [NASA OSD-944](https://osdr.nasa.gov/bio/repo/data/studies/OSD-944)

## Installation and Setup

To reproduce this analysis, clone this repository and set up a virtual environment. The required Python packages include `pandas`, `requests`, `matplotlib`, `seaborn`, `scipy`, and `scikit-bio`.

### Windows Setup
```bash
# Create a virtual environment
python -m venv venv

# Activate the environment
.\venv\Scripts\activate

# Install required dependencies
pip install pandas requests matplotlib seaborn scipy scikit-bio

```

### macOS/Linux Setup

```bash
# Create a virtual environment
python3 -m venv venv

# Activate the environment
source venv/bin/activate

# Install required dependencies
pip install pandas requests matplotlib seaborn scipy scikit-bio

```

## Pipeline Overview

The analysis is broken down into sequential Jupyter Notebooks. They must be executed in order, as downstream notebooks rely on the cleaned data matrices generated by the earlier scripts.

### 1. Data Wrangling & Acquisition

* **`1_clean_metadata.ipynb`**: Downloads and parses the raw ISA metadata archives. Filters for relevant sequencing parameters (Prep 3) and isolates environmental swabs from low-biomass controls.
* **`2_clean_assaydata.ipynb`**: Cleans sequencing quality metrics, extracting DNA concentrations, total read counts, and contamination percentages.
* **`3_download_annotations.ipynb`**: Automates the batch downloading of `gene-coverage-annotation-and-tax.tsv` files directly from the NASA OSDR AWS S3 buckets for the filtered target samples.

### 2. Matrix Construction

* **`4_build_master_matrix.ipynb`**: Concatenates all individual sample annotation files into a unified master functional matrix. Normalizes raw gene coverage into Counts Per Million (CPM) based on total sample read depth.

### 3. Functional and Statistical Analysis

* **`5_resistome.ipynb`**: Integrates the KEGG AMR (ko01504) and BacMet BRG databases. Identifies antimicrobial and biocide resistance genes, flags multidrug/dual-resistance determinates, and maps these traits to specific taxa (e.g., *Acinetobacter*, *Brevundimonas*).
* **`6_stat.ipynb`**: Computes alpha and beta diversity metrics across the physical cleanroom locations. Performs non-parametric statistical testing (Kruskal-Wallis) and PERMANOVA to determine spatial significance and community structure variance.
